[Big Data] Recommender System

Movie recommender system running on Hadoop. It is a chain of five map-reduce jobs running on hadoop. It is based on item-item Collaborative Filtering (item CF).

  • GitHub page: https://github.com/fanchi/RecommenderItemCF
  • Maven version: https://github.com/fanchi/RecommenderItemCFMaven

    • [Big Data] Page Rank

    Implementation of page rank on Hadoop. It has two map-reduce jobs. Given an initial page rank (weight) and transition matrix, it iteratively updates the page rank based on the previous page rank and transition matrix.
